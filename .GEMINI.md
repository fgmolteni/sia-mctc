# TITULO: SIA(Sistema Interno de Administración)

## Descripción:
Es un sistema de gestion de expedientes y calculo de viaticos para la administracion publica.
El sistema tiene un sistema de login, solo logeado con usuarios registrados en la base de datos por un usuario administrados.
La aplicacion debe contener varias pantallas:
    - ingreso: pantalla de ingreso.
    - agentes: vista de registro de usuarios y otorgar atributos de atributos.
    - reporte de anticipo: vista de generacion de reportes de anticipo de viaticos y tabla de viaticos generados.
    - pantalla de caja chica: registro de montos gastados y monto total sobrante.
